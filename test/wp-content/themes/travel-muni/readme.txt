== Description ==
Travel Muni is a feature-rich travel and tour booking WordPress theme to create SEO-friendly travel agency websites. Travel Muni is compatible with WP Travel Engine, the most popular travel booking plugin, which allows you to sell tours online. You can add tours, destinations, trip types, and activities and highlight your tours. In addition, it offers an excellent user experience to your guests to easily and quickly book any tour they like. Travel Muni is speed optimized, translation ready, and mobile-friendly. Check the demo at https://wptravelenginedemo.com/travel-muni/, documentation at https://docs.wptravelengine.com/docs/travel-muni/, and get support at https://wptravelengine.com/support/


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
